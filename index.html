📷 Insta360 Studio SOP
📷 Step 0 — Do this before clicking 🤖 AI in Pending Editing
This is the only desktop step. Export from Insta360 Studio → upload to Google Drive → paste link into Airtable Optimized Video Link → change status to Editing and Optimisation in Progress → click 🤖 AI. The Content Machine handles everything after that.
🎥 Full Episode — Long Form
16:9 landscape · 20 Mbps · Two .insv files
1
Import — yellow folder → select both VID_029.insv + VID_030.insv
2
Aspect ratio → confirm 16:9 (bottom-right of preview)
3
View Angle Gear (∨ chevron below preview) → select DEWARP (yellow) — always this, never MEGA/ULTRA/LINEAR
4
Default framing — Pan -210° · Tilt 0.0° · FOV 100° · Zoom 0.40 — usually leave as-is
5
Add 2–3 keyframes (+ button in toolbar) — near start, middle, end of timeline yellow dots lock framing
6
Play through — verify Kevin stays centred. If he drifts, drag that keyframe and adjust Pan
7
Export (yellow button top-right)

📹 Summary Clip — Short Form
9:16 portrait · 15 Mbps · ~37 seconds
1
Import VID_030.insv (the shorter clip, ~37s)
2
Aspect ratio → switch to 9:16 portrait (TikTok / Reels / Shorts)
3
View Angle Gear → DEWARP · Pan -195° · Tilt -32° · FOV 69.9° · Zoom 0.46
4
Trim — drag left handle to clean start, right handle to ~37s (orange × marker)
5
Add 2 keyframes — start + end of trimmed clip
6
Export

✅ After export — the AI handles everything else
1. Upload to Drive → 2. Paste link in Airtable → 3. Status → Editing in Progress → 4. Click 🤖 AI → Intro Editor → Subtitles → Thumbnail → Copywriting → ✅ Done
⚠️ Rules — never break these
• Always use DEWARP — never MEGA, ULTRA, or LINEAR
• Full video = 16:9 landscape · Summary = 9:16 portrait
• Always add at least 2 keyframes — start and end
• Export Full at 20 Mbps · Summary at 15 Mbps
• File names must include Ep number + topic
• LFMD clips come from the same file as the full video
